# Quản lý Website Pace

## Mô tả
Làm lại trang web Pace theo kiểu cơ bản

## Yêu cầu hệ thống
- Python 3.9.+ (dự án sử dụng Python 3.9.6)
- Django==4.2.7 (bắt buộc)
- Sử dụng Visual Studio Code
- có cài pip (cách cài python3 -m pip install django==4.2.7)

## Cách sử dụng:
- Mở project trên Visual Studio Code cd đến thư mục chứa dự án
- Mở terminal của dự án
- Kiểm tra đã cài Python và Django
- Cài thư viện dự án (dùng lệnh pip3 install -r requirements.txt)
- Không cần cài venv (môi trường ảo) vì đây là dự án demo basic có thể chạy trực tiếp, nếu muốn cài thì dùng lệnh 
source venv/bin/activate

## Chạy ứng dụng:
   Lần lượt dùng 3 lệnh
   - python3 manage.py makemigrations
   - python3 manage.py migrate
   - python3 manage.py runserver
Nhấp vào đuờng link trong terminal
